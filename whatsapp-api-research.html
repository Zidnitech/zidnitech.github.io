TL;DR

WhatsApp API cost has two independent layers: Meta's per-message fees (fixed for all providers) + the platform subscription (varies 146–750+ SAR/mo). Some providers add markup on Meta fees — others don't. A small business sending 1,000 messages/month typically pays 250–450 SAR total.

Layer 1 — Meta's Conversation Fees

These rates are set by Meta and are identical regardless of which Business Solution Provider (BSP) you use. Saudi Arabian numbers (as of 2026):

Message Type USD / message SAR / message Use case
Marketing $0.0455 ~0.1706 SAR Broadcasts, promotions, retargeting
Utility $0.0107 ~0.040 SAR Order confirmations, shipping updates
Authentication $0.0107 ~0.040 SAR OTPs, verification codes
Service (within 24h window) $0.00 Free Replies to user-initiated messages

Source: Meta Business Platform Pricing

The 24-Hour Free Window

When a user messages you first, you have 24 hours to reply at zero Meta cost. Every message you send during that window (confirmations, follow-ups, upsells) is free. After 24h, you need an approved Template, charged at the rates above.

⚠ Markup trap: Some providers add 10–30% on top of Meta's rates. Always confirm whether your BSP passes Meta fees directly or inflates them. Providers like Wati and Respond.io do not add markup.

The Cost Formula

Calculate your monthly cost
// Step 1: Meta fees
marketing_cost = marketing_messages × 0.1706 // SAR
utility_cost = utility_messages × 0.040 // SAR
free_replies = 0 // within 24h window

// Step 2: Platform subscription
platform_fee = 146–746 SAR // Wati Growth–Business

// Total
monthly_cost = marketing_cost + utility_cost + platform_fee

Key Cost Levers

1. Message type classification: A confirmation message with a promotional line gets classified as Marketing (0.1706 SAR) instead of Utility (0.040 SAR). That's 4× the cost per message. Audit your templates.

2. 24-hour window utilization: All service replies within 24h of a user message are free. Batch post-purchase messages (confirmation + shipping + review request) within that window.

3. Provider markup: At 5,000 marketing messages/month, a 20% markup = ~170 SAR/month = ~2,040 SAR/year in unnecessary cost.
